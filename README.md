# Arbol φ — Test del Árbol con Proporción Áurea

App web para el Test del Árbol (Tree Drawing Test) con análisis basado en la Proporción Áurea (φ).

Basado en el estudio de Stanzani Maserati & Zama (2026) publicado en *Scientific Reports* para la detección de deterioro cognitivo.

## Uso
1. Toma una foto o sube un dibujo de árbol
2. Marca 3 puntos: copa, base del tronco, unión tronco-copa
3. Calcula los índices φ y obtén la correlación diagnóstica

## Tecnología
HTML+CSS+JS puro — sin frameworks, sin servidor, sin dependencias.

**Nota:** Herramienta de investigación. No reemplaza evaluación clínica.
